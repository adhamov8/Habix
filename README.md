# Cohabit — Веб-сервис коллективных челленджей и трекинга привычек

Веб-сервис для совместного формирования привычек: создавайте челленджи, приглашайте участников по ссылке, отслеживайте прогресс через ленту активности и рейтинг.

**Стек:** Go · PostgreSQL · React + TypeScript · Docker Compose

---

## Быстрый запуск

**Требования:** Docker Desktop 

```bash
git clone https://github.com/adhamov8/Cohabit.git
cd Cohabit
cp .env.example .env
docker compose up --build
```

Готово. Открыть: **http://localhost:3000**

| Сервис | Адрес |
|--------|-------|
| Приложение | http://localhost:3000 |
| API + Swagger | http://localhost:8080/swagger/index.html |
| Метрики | http://localhost:8080/metrics |
| Prometheus | http://localhost:9090 |

## Основные возможности

- Создание челленджей с гибкими правилами: рабочие дни, максимум пропусков, дедлайн отметки
- Приглашение по ссылке — новый участник сразу попадает в челлендж после регистрации
- Ежедневные отметки с комментарием и фото, heatmap прогресса
- Лента активности, лайки, комментарии, лидерборд
- Система достижений (серии, процент выполнения)
- JWT-аутентификация с ротацией токенов, Prometheus-метрики
